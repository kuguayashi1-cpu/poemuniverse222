# 🌠 诗意星空 - 使用说明

## 📁 文件说明

- **index.html** - 主页面，包含输入和输出功能
- **overview.html** - 总览页面，展示所有历史记录
- **README.md** - 详细的项目说明文档

## 🚀 如何使用

### 方法一：直接打开
1. 双击 `index.html` 文件
2. 浏览器会自动打开诗意星空应用
3. 开始输入中文文本，点击"生成诗歌"按钮

### 方法二：通过浏览器
1. 打开任意浏览器（Chrome、Firefox、Edge等）
2. 按 `Ctrl+O` 或点击"文件" → "打开文件"
3. 选择 `index.html` 文件
4. 开始使用应用

## ✨ 功能特色

### 🌌 主页面功能
- **星空背景** - 动态星点粒子特效
- **中文输入** - 支持中文文本输入
- **诗意转换** - 自动将文本转换为诗歌格式
- **漂浮文字** - 生成随机漂浮字符效果
- **历史记录** - 自动保存转换历史
- **保存功能** - 可下载诗歌文本文件

### 📚 总览页面功能
- **历史展示** - 以卡片形式展示所有历史记录
- **字体调节** - 可调整诗歌字体大小
- **颜色切换** - 支持多种文字颜色
- **详情查看** - 点击卡片查看完整诗歌
- **复制下载** - 支持复制和下载功能

## 🎨 界面说明

### 主页面布局
1. **顶部标题** - "诗意星空"大标题，带呼吸动画
2. **输入区域** - "故乡入口"文本输入框
3. **输出区域** - 左侧诗意结果，右侧漂浮文字
4. **操作按钮** - 查看总览、保存结果

### 总览页面布局
1. **控制面板** - 字体大小、颜色选择
2. **历史网格** - 卡片式展示所有记录
3. **浮动文字** - 背景中的动态文字效果

## 🔧 技术特点

- **纯HTML/CSS/JavaScript** - 无需安装任何软件
- **响应式设计** - 支持手机、平板、电脑

1. **输入建议** - 输入完整的中文句子，效果更佳

## 🆘 常见问题

**Q: 为什么没有历史记录？**
A: 首次使用需要先生成一些诗歌，历史记录会自动保存。

**Q: 可以保存多少条历史？**
A: 最多保存10条最新记录，超出会自动删除最旧的。

**Q: 支持哪些浏览器？**
A: 支持所有现代浏览器，建议使用Chrome或Firefox获得最佳体验。

**Q: 数据会丢失吗？**
A: 数据保存在浏览器本地，清除浏览器数据会丢失历史记录。
